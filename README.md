# The new sound of politics
Repository for the research project of UCD Politics and Data Science module Quantitative Text Analysis. The goal is to examine how language in political bodies has changed due to the possible influence of social media on politicians tactical considerations.

This repor includes all code and scripts that can be used to replicate the findings.

To replicate analysis run those scripts that start with a number and an underscore in ascending order.
Important! Begin with 2_data_wrangling_dip_api.r to not pull new data from the API.

Render the quarto document index.qmd to get the full paper.
